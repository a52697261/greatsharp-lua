# Events

## List of events:

### OnInit

Fired when script init.

## OnFastTick

Fired when game update\(every frame\).

## OnTick

Fired when game update\(30 per/second\).



### OnDraw

Fired every time the game renders a frame . Can be used to draw to the screen using the Screen.\* functions
